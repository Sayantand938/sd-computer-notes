# C Programming

## Introduction

C is a general-purpose programming language developed by Dennis Ritchie at Bell Labs in the early 1970s. It is widely used for system programming, embedded systems, and application development.

## Features of C

* Simple and efficient
* Fast execution speed
* Portable across different platforms
* Supports structured programming
* Provides low-level memory access

## Basic Structure of a C Program

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

## Data Types

Common data types in C include:

| Data Type | Description                             |
| --------- | --------------------------------------- |
| int       | Integer numbers                         |
| float     | Decimal numbers                         |
| char      | Single characters                       |
| double    | Double-precision floating-point numbers |

## Control Statements

C provides several control statements:

* `if` and `else`
* `switch`
* `for`
* `while`
* `do-while`

Example:

```c
if (age >= 18) {
    printf("Adult");
} else {
    printf("Minor");
}
```

## Functions

Functions help organize code into reusable blocks.

```c
int add(int a, int b) {
    return a + b;
}
```

## Advantages of C

1. High performance
2. Easy to learn
3. Extensive library support
4. Widely used in operating systems and embedded devices

## Conclusion

C remains one of the most important programming languages. Learning C helps programmers understand computer systems and provides a strong foundation for learning other programming languages.
